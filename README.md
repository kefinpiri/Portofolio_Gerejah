

 Deskripsi
Website ini merupakan proyek pengembangan web yang dibuat untuk menampilkan informasi gereja secara online.  
Dibangun menggunakan HTML, CSS, dan JavaScript dengan tampilan yang sederhana dan mudah digunakan.



Berisi informasi tentang gereja seperti:
- Sejarah gereja
- Visi dan misi
- Struktur organisasi


Menampilkan foto-foto kegiatan gereja seperti:
- Ibadah
- Kegiatan jemaat
- Acara khusus

Berisi informasi jadwal kegiatan gereja, seperti:
- Jadwal ibadah mingguan
- Kegiatan pelayanan
- Event khusus


Menyediakan informasi untuk menghubungi gereja:
- Alamat
- Nomor telepon
- Media sosial
